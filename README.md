### Gerenciador de tarefas em Python
Criei um programa básico no Pycharm utilizando a linguagem python para demonstrar meus conhecimentos. Nele o usuário pode visualizar a lista, adicionar tarefas, marcar tarefas como concluídas e encerrar o programa

## Funcionalidades
- Mostra a lista completa para o usuário
- O usuário pode adicionar tarefas
- O usuário pode marcar as tarefas como concluídas
- O usuário pode encerrar o programa

## Conceitos utilizados
- Listas
- Input (entrada de dados)
- Enumerate (função para numerar a lista)
- If e else (condicionais)
- While e for (loops)
